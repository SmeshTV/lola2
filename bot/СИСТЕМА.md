# 📊 Система XP и грибов — Документация

## 🎯 За что даётся XP и грибы

### 🎧 За сидение в голосовом канале

| Награда | Количество | Интервал | Описание |
|---|---|---|---|
| **Discord XP** | +1 | каждые 60 секунд | XP для Discord уровня |
| **Грибы (🍄)** | +1 | каждые 900 секунд (15 мин) | Валюта для сайта |

**Пример:**
- Сидишь в voice 15 минут → **+15 XP** и **+1 гриб** 🍄
- Сидишь 1 час → **+60 XP** и **+4 гриба** 🍄
- Сидишь 4 часа → **+240 XP** и **+16 грибов** 🍄

### 💬 За сообщения в чате

| Награда | Количество | Кулдаун | Описание |
|---|---|---|---|
| **Discord XP** | +1-3 (случайно) | каждые 180 секунд (3 мин) | XP для Discord уровня |
| **Грибы** | 0 | — | За сообщения грибы НЕ даются |

**Пример:**
- Написал 10 сообщений (с интервалом 3мин+) → **+10-30 XP**
- Грибы за сообщения не начисляются — только за voice и игры на сайте

### 🎮 За игры на сайте

| Игра | Победа | Поражение | Ничья |
|---|---|---|---|
| КНБ (ставка 10🍄) | +10 🍄 | -10 🍄 | 0 🍄 |
| Казино (ставка 50🍄) | +50 🍄 | -50 🍄 | — |

## 📐 Формулы уровней

### Discord XP → Discord Level
Каждый уровень = **level × 100 XP**

| Уровень | XP нужно | Всего XP |
|---|---|---|
| 1 → 2 | 100 | 100 |
| 2 → 3 | 200 | 300 |
| 3 → 4 | 300 | 600 |
| 4 → 5 | 400 | 1000 |
| 5 → 6 | 500 | 1500 |

### Сайт XP → Сайт Level
Каждый уровень = **level × 200 XP**

| Уровень | XP нужно | Всего XP |
|---|---|---|
| 1 → 2 | 200 | 200 |
| 2 → 3 | 400 | 600 |
| 3 → 4 | 600 | 1200 |
| 4 → 5 | 800 | 2000 |

## 🔧 Как изменить настройки (редактирование)

Открой файл `bot/src/index.js`, найди секцию `CONFIG`:

```javascript
const CONFIG = {
  // ... другие настройки ...
  
  // XP за голосовой канал
  voiceXpInterval: 60,        // СЕКУНД для +1 XP (Discord XP)
  voiceXpAmount: 1,           // Сколько XP давать

  // Грибы за голосовой канал
  voiceMushroomInterval: 900, // СЕКУНД для +1 гриб (900 = 15 мин)
  voiceMushroomAmount: 1,     // Сколько грибов давать

  // XP за сообщения
  messageXpRange: [1, 3],     // Случайный XP (мин, макс)
  messageXpCooldown: 180,     // СЕКУНД кулдаун между сообщениями (180 = 3 мин)

  // Синхронизация
  syncInterval: 300000,       // Миллисекунды (300000 = 5 мин)
};
```

### Примеры изменений:

**Больше грибов за voice:**
```javascript
voiceMushroomInterval: 60,   // каждые 60 сек вместо 120
voiceMushroomAmount: 2,      // +2 гриба вместо +1
```

**Больше XP за voice:**
```javascript
voiceXpInterval: 30,         // каждые 30 сек вместо 60
voiceXpAmount: 2,            // +2 XP вместо +1
```

**Больше XP за сообщения:**
```javascript
messageXpRange: [5, 15],     // +5-15 XP вместо +1-3
messageXpCooldown: 60,       // кулдаун 60 сек вместо 30
```

## 🏗 Архитектура (как это работает)

```
┌──────────────────┐              ┌──────────────┐
│  DISCORD         │              │   SUPABASE    │
│  БОТ (Node.js)   │              │   (База)      │
│                  │              │               │
│  Voice: +1 XP/60с│── flush ──→ │  users table  │
│  Voice: +1🍄/120с│  (5 мин)    │  ├─ xp         │
│  Msg: +1-3 XP/30с│              │  ├─ level      │
└──────────────────┘              │  ├─ discord_xp │
                                  │  ├─ discord_lvl│
┌──────────────────┐              │  ├─ mushrooms  │
│  САЙТ (React)    │              │  └─ ...        │
│                  │ ←── read ──  └──────────────┘
│  Dashboard:      │
│  🎮 Сайт уровень │
│  💬 Discord ур.  │
│  🍄 Грибы        │
└──────────────────┘
```

### Поток данных:
1. **Бот** собирает XP и грибы → буфер
2. Каждые **5 минут** → буфер отправляется в Supabase
3. **Сайт** читает из Supabase → показывает на Dashboard
4. Кнопка **↻** на Dashboard → обновляет данные

## 📋 Команды бота

| Команда | Описание |
|---|---|
| `/rank` | Показать ранг и XP |
| `/leaderboard` | Топ-10 игроков |
| `/link` | Привязать Discord к сайту |
| `/sync` | Принудительно отправить буфер (админ) |

---

## 🗄️ SQL функции в Supabase

### `add_site_xp(user_id, xp_amount, messages)`
Начисляет Discord XP → обновляет `discord_xp` и `discord_level`

### `add_mushrooms_voice(user_id, mushrooms)`
Начисляет грибы → обновляет `mushrooms`

### `add_mushrooms(user_id, amount)`
Начисляет грибы для игр на сайте (КНБ, казино и т.д.)

---

## ⚠️ Важные заметки

1. **Бот должен быть запущен** — иначе XP не начисляется
2. **Привязка через `/link`** — нужна хотя бы один раз
3. **Минимальное время** — нужно минимум 60с в voice для первого XP
4. **Буфер** — XP записывается раз в 5 минут (или через `/sync`)
