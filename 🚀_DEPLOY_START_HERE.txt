✅ ГЛАВНОЕ ПОМНИТЬ:

  ✅ git push → Netlify сам всё соберет и задеплоит
  ✅ Переменные добавляй в Netlify UI, не в .env
  ✅ dist/ не коммитить (она в .gitignore)
  ✅ Тестируй локально перед push

❌ Не делай:

  ❌ Не коммитьте .env файл
  ❌ Не коммитьте node_modules
  ❌ Не трогайте dist/ вручную
  ❌ Не забывайте про переменные окружения

💬 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

  1. Проверь DEPLOYMENT.md раздел "🐛 Если что-то не работает"
  2. Посмотри логи на Netlify (Site → Deploys → последний deploy → Build log)
  3. Проверь консоль браузера (F12) на ошибки
  4. Убедись что переменные окружения установлены (NETLIFY_ENV_SETUP.md)
